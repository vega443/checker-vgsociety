# &vgsociety

## Installation

1. Installer Python 3.10+
2. Double-cliquer sur `setup.bat`
3. Editer `.env`, `tokens.txt`, `proxies.txt`
4. Double-cliquer sur `lance.bat`

## Fichiers

- `.env` :: configuration (webhooks, threads, etc.)
- `tokens.txt` :: 1 token par ligne
- `proxies.txt` :: 1 proxy par ligne (`ip:port`)
- `hits.txt` :: usernames trouves
